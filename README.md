# Travel Website 🌍✈

This is a responsive *Travel Website* project created using *HTML, CSS, and JavaScript*.  
The website allows users to explore different travel destinations, get information about activities, and experience a beautiful design.

## 🚀 Live Demo
[Click here to view the website](https://urvashikhatri1201-droid.github.io/travel-website/)

## 📌 Features
- Responsive design (works on mobile, tablet, desktop)
- Attractive hero section with a call-to-action button
- Sections for destinations, activities, and contact
- Clean and modern UI with CSS styling

## 🛠 Technologies Used
- HTML
- CSS
